# Basit Person App (React + Dexie + IndexedDB)

Bu küçük örnek:
- Tek model: Person { id, name }.
- IndexedDB (Dexie) kullanıyor — veriler tarayıcı profilinde diskte saklanır ve tarayıcı kapansa da kalıcıdır.

Hızlı kurulum:
1. Proje oluştur (vite + react-ts):
   npm create vite@latest my-app -- --template react-ts
   cd my-app
2. Dexie yükle:
   npm install dexie
4. Çalıştır:
   npm run dev
